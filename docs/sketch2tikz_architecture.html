Nine-agent orchestration pipeline
1 · Image processing agent
Cleaning, denoising, contrast & edge detection
2 · Vision understanding agent
IBM Granite Vision
Shapes, nodes, arrows, OCR, layout
3 · RAG retrieval agent
Vector DB · embeddings · templates
4 · TikZ generation agent
IBM Granite Code
Nodes, paths, styles, LaTeX libraries
5 · TikZ validation agent
Syntax, brackets, packages, node refs
6 · Compilation agent
Runs pdflatex → PDF, SVG, PNG
7 · Self-healing agent
Detect → explain → repair → retry loop
8 · Quality review agent
Sketch vs. output similarity & confidence
9 · Export agent
.tex · .pdf · .svg · .png · .zip
Agents execute in a sequential pipeline (1→9) with a self-healing feedback loop between validation/compilation and repair, orchestrated end-to-end by IBM watsonx Orchestrate.
